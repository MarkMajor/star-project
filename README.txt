
The Stars Project - J. Wallin

This is a project to plot star data for astrolabes,
starwheels, star maps, constellatoin maps, and other
astronomical projections.  The project uses data from 
several sources, including:

The Bright Star Catalog
The Constellation Boundary map - augmented to with additioanl points
The Constellation outline file
A list of bright stars
